プライバシーポリシー

最終更新日：2026年5月23日

「もう限界？」をご利用いただきありがとうございます。私たちは、できるだけデータを端末内に残すことを重視しています。

1. 情報の収集と利用

ローカル優先：吐き出し本文、タグ、混乱レベル、履歴は端末内に保存されます。私たちのサーバーへ吐き出し内容をアップロードしません。

2. 保存とバックアップ

記録は端末内に保存され、iOSのバックアップに含まれる場合があります。バックアップなしでアプリを削除すると、復元できない可能性があります。

3. 広告と第三者パートナー

アプリはGoogle AdMobとGoogle User Messaging Platform（UMP）を統合しています。広告は現在、任意で開く応援ページに表示されます。アプリ起動時に、UMPを通じて広告同意状況を確認または更新する場合があります。Google Mobile Ads SDKは、同意状況により広告リクエストが許可された後にのみ起動します。

Googleおよび広告パートナーは、端末識別子、広告データ、IPアドレス、IPアドレスから推定されるおおよその位置、広告またはプロダクト操作、クラッシュデータ、診断情報、パフォーマンスデータなど、広告に関連する情報を端末から収集または受け取る場合があります。これらの情報は、第三者広告、私たちの広告またはマーケティング、分析、アプリ機能、不正や悪用の防止、適用法令への対応に使用されることがあります。Googleは、自社のポリシーおよび端末またはアカウントのプライバシー設定に従ってこれらの情報を処理します。

このアプリは、吐き出し本文、タグ、混乱レベル、落ち着きセッション履歴、ローカルの吐き出し履歴をGoogleまたはその他の広告サービスへ送信しません。

このアプリはiOSの位置情報許可を求めず、正確なGPS位置情報にもアクセスしません。広告に関するおおよその位置は、GoogleがIPアドレスなどのネットワーク情報から推定するものです。

広告トラッキングとパーソナライズは、iOSのプライバシー設定、App Tracking Transparencyの確認、および地域で利用可能なGoogleの同意または広告パーソナライズ設定から管理できます。必要な場合、アプリは設定画面と任意の応援ページにプライバシー設定への入口を表示します。

4. 外部リンク

Apple Mapsなど外部サービスを開く場合があります。アプリを離れた後は、各サービスのプライバシーポリシーが適用されます。
